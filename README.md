# join_ad

### How to use for Systemd OS :

Copy and Past in your terminal :

```bash
wget -Nnv https://raw.githubusercontent.com/liberodark/join_ad/master/install.sh && chmod +x install.sh; ./install.sh
```

### Linux Compatibility : (Systemd)

- Deepin 15.x
- Ubuntu 16.04 / 19.10
- Centos 7.x / 8.x
- Oracle 7.x
- Red Hat 7.x / 8.x
